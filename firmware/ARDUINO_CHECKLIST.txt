PHASE 1: SOFTWARE SETUP
□ Download & Install Arduino IDE 2.x
□ Add ESP32 Board Manager URL in Preferences
□ Install ESP32 board support (v2.0.14+)
□ Install NimBLE-Arduino library
□ Install ArduinoJson library
□ Download & install ESPAsyncWebServer.zip
□ Download & install AsyncTCP.zip

PHASE 2: HARDWARE CHECK
□ ESP32 connected via USB
□ Power LED lights up
□ All 12 switches installed
□ Both encoders installed
□ Diodes installed (cathode to key)
□ No short circuits
□ Pin connections verified:
  □ Rows: GPIO 16, 17, 18
  □ Cols: GPIO 21, 22, 23, 19
  □ Enc1: A=32, B=33, SW=27
  □ Enc2: A=25, B=26, SW=13

PHASE 3: BOARD CONFIGURATION
□ Tools → Board: "ESP32 Dev Module"
□ Tools → Upload Speed: 921600
□ Tools → Partition Scheme: "Huge APP (3MB No OTA/1MB SPIFFS)"
□ Tools → Port: Correct COM port selected

PHASE 4: UPLOAD FIRMWARE
□ Open Micropad.ino
□ Click Verify (✓) - No errors
□ Click Upload (→)
□ Upload successful (see "Hard resetting via RTS pin...")

PHASE 5: SERIAL MONITOR TEST
□ Open Serial Monitor (115200 baud)
□ Press RESET button on ESP32
□ See boot messages:
  □ "Micropad Firmware 1.0.0"
  □ "Matrix initialized"
  □ "Encoder initialized" (both)
  □ "Profile loaded: General"
  □ "Micropad ready!"
□ No error messages

PHASE 6: KEY TESTING
□ K1 pressed → See "Key 0 pressed"
□ K2 pressed → See "Key 1 pressed"
□ K3 pressed → See "Key 2 pressed"
□ K4 pressed → See "Key 3 pressed"
□ K5 pressed → See "Key 4 pressed"
□ K6 pressed → See "Key 5 pressed"
□ K7 pressed → See "Key 6 pressed"
□ K8 pressed → See "Key 7 pressed"
□ K9 pressed → See "Key 8 pressed"
□ K10 pressed → See "Key 9 pressed"
□ K11 pressed → See "Key 10 pressed"
□ K12 pressed → See "Key 11 pressed"

PHASE 7: ENCODER TESTING
□ Encoder 1 CW → See "Encoder 1 turned: 1"
□ Encoder 1 CCW → See "Encoder 1 turned: -1"
□ Encoder 1 Press → See "Encoder 1 pressed"
□ Encoder 2 CW → See "Encoder 2 turned: 1"
□ Encoder 2 CCW → See "Encoder 2 turned: -1"
□ Encoder 2 Press → See "Encoder 2 pressed"

PHASE 8: PROFILE TESTING
□ Hold K1 + K4 (1 sec) → See "Switched to profile 1"
□ Hold K1 + K12 (1 sec) → See "Switched to profile 0"

PHASE 9: BLUETOOTH PAIRING
□ Windows Settings → Bluetooth & devices
□ Click "Add device" → Bluetooth
□ "Micropad" appears in list
□ Click "Micropad"
□ Device connected
□ Serial shows: "BLE HID client connected"

PHASE 10: HID FUNCTION TEST
□ Open Notepad
□ Type "Hello World"
□ Select all (Ctrl+A)
□ Press K1 → Text copied
□ Press K2 → Text pasted
□ Open YouTube/Spotify
□ Press K10 → Play/Pause works
□ Rotate Encoder 1 → Volume changes
□ Press Encoder 1 → Mute works

PHASE 11: FINAL VERIFICATION
□ All keys work correctly
□ Both encoders responsive
□ Profile switching works
□ BLE connection stable
□ No serial errors
□ 10-minute stress test passed

Common Issues & Solutions:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Upload Failed → Hold BOOT button during upload
Keys Not Working → Check matrix wiring & diodes
Encoder Wrong Direction → Swap A and B pins
BLE Won't Pair → Remove old device, restart ESP32
Brownout Error → Use powered USB hub
